# bus-time
Live bus time update using TransportAPI

## Dependencies
requests
feedparser
Pillow

##### Instruction 
1) Set bus number on bus_time.py line 20 
2) Get token from https://transportapi.com, and amend bus_time.py line 21
3) Run bus_time.py
